INSTRUCCIONES:
=============
1) la carpeta wavs debe ponerse en C:/
2) Usar LogicTouch Layout en Touch OSC APP

BOTONES:
========
1) VOL: Modular el volumen del track principal
2) PAN: Cambia el pitch del track principal
3) Cycle: loopea el track principal en un lapso de 3 segundos
4) OFF: cancela el efecto de Cycle
5) Read: reproduce el primer track
6) Latch: reproduce el track grabado
7) by Marker: reproduce el cuarto track
8) Touch: reproduce un track secundario de un grito
9) Click: reproduce un track terciario de una base de techno
10) ^ : Avanza al siguiente track
11) v : Retroce al track anterior
12) RECORD: inicia una grabación
13) R: para la grabación
14) I: importa un wav desde el PC
15) S: pone distorsión aleatoria al track principal
16) M: mutea el track principal sin detenerlo
17) Stop: detiene el track principal
18) Play: reanuda el track principal
19) >>: adelanta el track principal
20) <<: rebobina el track principal
